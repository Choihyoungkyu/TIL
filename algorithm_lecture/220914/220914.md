# 이산 수학



#### 조건 명제 (p -> q)

|  p   |  q   | p -> q |
| :--: | :--: | :----: |
|  T   |  T   |   T    |
|  T   |  F   |   F    |
|  F   |  T   |   T    |
|  F   |  F   |   T    |

- p가 F이면 무조건 T



#### 쌍방조건명제 (p <-> q)

|  p   |  q   | p <-> q |
| :--: | :--: | :-----: |
|  T   |  T   |    T    |
|  T   |  F   |    F    |
|  F   |  T   |    F    |
|  F   |  F   |    T    |



#### 조건명제의 역, 이, 대우

- 역 : q -> p
- 이 : ~p -> ~q
- 대우 : ~q -> ~p

|  p   |  q   | p -> q | q -> p(역) | ~p -> ~q(이) | ~q -> ~p(대우) |
| :--: | :--: | :----: | :--------: | :----------: | :------------: |
|  T   |  T   |   T    |     T      |      T       |       T        |
|  T   |  F   |   F    |     T      |      T       |       F        |
|  F   |  T   |   T    |     F      |      F       |       T        |
|  F   |  F   |   T    |     T      |      T       |       T        |





#### 귀류법

- 명제의 참을 증명하는 방법
- 그 명제의 부정을 참이라 가정하고 진행해서 거짓인걸 밝힘
- ex) ~(p -> q) = p A ~q 를 증명해도 됨
